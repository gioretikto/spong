# spong

A pong clone

## Dependencies

You will need:

  -  SDL 2.0
  -  SDL Mixer 2.0
  -  SDL TTF 2.0


sudo apt-get install libsdl2-dev libsdl2-mixer-dev libsdl2-ttf-dev

## Run the game

To run the game:

`./spong`

## Usage

- Return to start/pause the game

- F11 to toggle full screen mode

- Esc to exit
